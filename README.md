# ファームウェア

QMK Firmwareで用意しています。
ソースコードは[こちら](https://github.com/takashicompany/qmk_firmware/tree/master/keyboards/takashicompany/klec_01)です。

RemapにVIAファームウェアを登録しておりますので、[こちら](https://remap-keys.app/catalog/Tp7FO3o6xTwo2X9V9IJW/firmware)から書き込みが可能です。
キーマップの変更も[Remap](https://remap-keys.app/)から可能です。
